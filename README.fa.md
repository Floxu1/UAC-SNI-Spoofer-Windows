## معرفی برنامه

UAC Spoofer Desktop یک داشبورد شبکه برای ویندوز است که بر پایه Xray و روش SNI Spoofing گروه Patterniha طراحی شده است.

این برنامه مسیرهای مخصوص هر اپراتور را مدیریت می‌کند، جفت‌های SNI و Edge را بررسی و رتبه‌بندی می‌کند، پروکسی ویندوز را به‌صورت خودکار اعمال می‌کند و هنگام قطع اتصال، خروج از برنامه یا راه‌اندازی مجدد، تنظیمات قبلی پروکسی را بازمی‌گرداند.

پروفایل‌های دسکتاپ برای **همراه اول / MCI** و **ایرانسل / IranCell** به‌صورت جداگانه تنظیم شده‌اند؛ بنابراین تغییر تنظیمات یک اپراتور باعث تغییر یا اختلال در تنظیمات اپراتور دیگر نمی‌شود.

## قابلیت‌ها

- حالت‌های جداگانه برای همراه اول، ایرانسل و Auto
- تونل محلی Xray با پشتیبانی از SOCKS و HTTP
- اسکنر SNI و Edge بر پایه روش Patterniha با نمایش زنده نتایج
- بهینه‌سازی شروع TLS و آماده‌سازی مسیر YouTube برای همراه اول
- پیشنهاد بهترین کانفیگ بر اساس تست واقعی صفحه و دانلود محدود
- رابط کاربری فارسی و انگلیسی با پشتیبانی صحیح از RTL و LTR
- قابلیت App Bypass
- نمایش لاگ زنده
- بررسی IP عمومی
- تنظیمات پیشرفته برای بهینه‌سازی اتصال
- بررسی به‌روزرسانی از طریق GitHub Releases
- بازگردانی ایمن تنظیمات پروکسی ویندوز پس از قطع اتصال، خروج یا راه‌اندازی مجدد برنامه

## اجرای نسخه Portable

1. فایل زیر را از بخش GitHub Releases دانلود کنید:

```text
UAC-Spoofer-Desktop-v1.5.0-Windows-x64.zip
```

2. فایل ZIP را به‌صورت کامل Extract کنید.
3. پوشه `_internal` باید در کنار فایل EXE باقی بماند.
4. فایل زیر را اجرا کنید:

```text
UAC-Spoofer-Desktop.exe
```

5. در صورت نمایش پیام UAC ویندوز، آن را تأیید کنید.

برای اجرای نسخه Portable نیازی به نصب Python نیست.

## اجرای برنامه از سورس

نیازمندی‌ها:

- Windows 10/11 نسخه 64 بیتی
- Python 3.11 یا 3.12

```powershell
python -m pip install -r requirements.txt
powershell -ExecutionPolicy Bypass -File .\install-engine.ps1
python main.py
```

## Carrier Mode

برنامه دارای چند حالت اپراتور است:

- `mci`: تنظیمات مستقل برای همراه اول، شامل Route، SNI، Edge، تست‌ها و بهینه‌سازی‌های اختصاصی.
- `irancell`: تنظیمات و مسیر مستقل برای ایرانسل.
- `auto`: پروفایل عمومی و خودکار برنامه.

برای دریافت نتیجه پایدارتر، بهتر است اپراتور فعلی را به‌صورت دستی انتخاب کنید.

## تنظیم بخش به‌روزرسانی برنامه

پیش از انتشار پروژه در GitHub، فایل زیر را باز کنید:

```text
uac_desktop/app_config.py
```

این مقادیر مربوط به بخش به‌روزرسانی هستند:

```text
UPDATE_REPOSITORY_URL
GITHUB_RELEASES_URL
LATEST_VERSION_URL
UPDATE_CHECK_ENDPOINT
PORTABLE_DOWNLOAD_URL
```

شماره نسخه برنامه نیز در فایل زیر قرار دارد:

```text
uac_desktop/__init__.py
```

متغیر نسخه:

```text
__version__
```

پس از قرار دادن پروژه در GitHub، یک Release ایجاد کنید؛ برای مثال:

```text
v1.5.0
```

سپس فایل ZIP نسخه Portable ویندوز را در همان Release آپلود کنید.

بررسی نسخه جدید از طریق فایل زیر انجام می‌شود:

```text
uac_desktop/update_checker.py
```

تابع مربوطه:

```text
check_latest_release()
```
